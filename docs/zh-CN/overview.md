# 00. 欢迎来到智能体世界

**本章内容：什么是 AI 智能体？为什么要搭建一个？这个指南会教你什么？**

## 什么是 AI 智能体？

想象你有一个**非常聪明但非常听话的私人助理**。

你对它说：*"帮我订一个晚上 7 点 4 个人的晚餐位，然后查一下我的日历看那时候有没有空，如果有空的话把餐厅地址发给朋友们。"*

普通聊天机器人只会**回答**你该怎么做。但**智能体**会真的去**做**——一步一步，使用工具，记住上下文，遇到问题自己处理。

> **智能体 = LLM（大脑）+ 工具（手脚）+ 记忆（笔记本）+ 规划（策略）**

## 真实世界的例子

| 做什么 | 以前 | 有了 AI 智能体 |
|---|---|---|
| 客服 | 人工看工单、查资料、回复 | 智能体看历史、查知识库、起草回复 |
| 代码审查 | 程序员看 diff、手动跑检查 | 智能体分析 diff、跑测试、找 bug |
| 研究助手 | 人工搜索、阅读、总结 | 智能体搜索、阅读、综合结论 |
| 个人自动化 | 写定时任务、脚本 | 智能体理解意图、自己规划步骤 |

## 你要搭建什么？

你从这里开始：

```python
❌ 一个只能做你硬编码事情的程序
```

最终变成这样：

```python
✅ 一个能自己想办法把事情做完的智能体
```

## 你需要准备什么

- ✅ **Python 3.9+** — [点此下载](https://python.org/downloads)
- ✅ **一个文本编辑器** — VS Code、PyCharm 甚至记事本都行
- ✅ **会用终端** — 会敲命令就行
- ✅ **API 密钥** — 我们先用免费或便宜的模型
- ✅ **好奇心** — 这个最重要

> **没有机器学习经验？完全没问题。** 你不需要训练模型，不需要数学公式。只需要按步骤来，逻辑思考就行。

### 学习建议

1. **自己敲代码** — 不要复制粘贴。打字能加深记忆。
2. **多尝试** — 改个参数、弄坏它、看看会发生什么。
3. **做练习** — 是为了巩固理解而设计的。
4. **卡住很正常** — 智能体很复杂。重读、搜索、提问。
5. **边学边搭自己的智能体** — 找一个你感兴趣的小项目，跟着每章内容实践。
